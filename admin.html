Lås upp

Ange ditt lösenord för att låsa upp admin-läget på den här enheten.

Första gången: klistra in en GitHub-token och välj ett lösenord. Tokenen krypteras med lösenordet och sparas bara i den här webbläsaren — ändringar skyddas på riktigt av GitHubs behörigheter, inte av lösenordet i sig.

Så skapar du tokenen (en gång)
  1. Gå till github.com → Settings → Fine-grained tokens
  2. Repository access: Only select repositories → hboll
  3. Permissions: Contents: Read and write samt Actions: Read and write
  4. Skapa och klistra in här.

Cuper

Ändringar committas till GitHub och syns på sajten inom någon minut. Turnerings-ID hittas i källkoden på cupens resultatsida (”tournamentId: …”). ProCup-cuper anger dataUrl i stället för ID. Lat/lon (valfritt) ger väderikoner på kommande matcher — slå upp ortens koordinater på t.ex. openstreetmap.org.
